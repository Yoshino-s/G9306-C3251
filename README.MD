- 关于上海居委/学校出行证明开具注意事项：

  火车站不查离沪证明！只需要48小时以内核酸+抗原+车票

  **5月8日个人经历就是需要接收地一切证明，居委才给你开让你走的证明，**这个证明是让你出门的。目前上海其实是没有政策让你可以走的，你除非真的是直系奔丧等等一些没有办法拒绝的理由，而且也需要证明，一般来说，如果你自驾的话，你高速公路也会上不去的。

## 连云港下车等待中转

从上海出发到连云港的是有两个专门车厢的，这两个车厢的人下车后由大白统一带到专门的候车区，安置当天从上海来连云港的旅客。别出站！站内闭环中转。建议预留2H以上中转。

需要中转乘坐的车次到了，就会放这一车次的人去候车上车，到达一班就放一班人上车中转，在连云港站没有要求看任何相关证明，也不看核酸结果，你自己车次到了注意上车就行。站内没饭吃！

5.9 连云港官方电话回答不能隔夜了，实操是二楼候车大厅乖乖过夜

### 下车等候

下车后在站台上别走，会有人安排，排队，清点人数，会收身份证，跟着队伍走就行，最好别问工作人员问题，他们自己内部也会吵架。需要走到登记的地方。        

### 三个登记

![image-20220510230530270](https://cdn.yoshino-s.online//typora_img/image-20220510230530270.png)

1. 问你是否是中转以及联系方式

注意自己带笔，否则和别人共用一支笔

2. 承诺书

内容包括承诺人姓名、身份证号、联系电话、从哪来（火车班次和座位号）、去哪里（火车班次、时间和座位号）、填写日期

3. 需要扫场所码，地址写

   ```
   到达江苏后居住地区：
   	南京市，雨花台区，雨花街道
   到达江苏后详细地址：
   	南京南站        
   ```

### 核酸检测

![image-20220510230600665](https://cdn.yoshino-s.online//typora_img/image-20220510230600665.png)

1. 核酸检测前需要扫码登记：其中有一项是需要登记地址，不是填在南京居住酒店的地址，而是最终目的地的地址！！！也就是你从南京第二天坐高铁到哪里去的地址。
2. 核酸结果查阅：核酸结果可以通过做核酸时扫核酸的界面以及苏康码查阅。核酸结果出来较快。

### **到达酒店**

车站到酒店路程大概十分钟，到达酒店门口后，先别下车，会有警察点名，告知注意事项。

大概率全季（软件大道地铁站店），酒店不能点外卖，但会提供食品补给。

5/9日晚安排南站附近的锦江之星南京火车南站店

补给物资：泡面在一楼大厅，水在四楼走廊，饮料在六楼楼梯口。前台没有充足人力给送水和吃的！所以大家还是自行准备一点食品物资带在路上。

注意事项：

1. 不准点外卖，不准串门。
2. 前台登记信息主要登记来的车次，第二天早上乘坐的车次和出发时间。补给物资也在楼下领。
3. 买的第二天8:00前的高铁需要早上5:50在楼下退房集合。
4. 进出酒店会有人喷消毒水，房间里面也有消毒水。
5. 不能开空调，可以开窗。
6. 实在要点外卖，第二天去高铁站候车室可以点外卖，亲测可行，详细操作可参考下文外卖注意事项。

### 第二天酒店出发

至于早上会有很多人跟你们一起走，分时间段送到南京南车站。

到南京南车站后会有大白大概提前一小时左右带你们去候车，**如果40分钟左右还没叫到你，一定要主动跟大白进行沟通**，让大白找到你的身份证（同一车次的身份证放到一起，前一天收上去的）以免错过自己车辆的出发时间。

### 关于外卖

1. 大家都要在南京南车站等待差不多两个小时，所以一点都不着急点外卖，麦当劳20分钟内能送达，且最适应大家的胃。
2. 地址写 南京南站-1F出入口南（1F出入口南里面）（一说 南京南站铁路到达（东）六号门） 要在那发呆2小时的，电话写卖，酒店是不允许点。外卖拿到后千万别在中转地方吃外卖，带到酒店去吃。有领导见到会抓。工作人员其实都不想大家点外卖，不确定就兜圈子问，半个小时能走吗类似的问题。
3. 建议下车前或者刚下车就点了，否则配送时间非常悬（除了站内麦当劳速度快），而且这个时间点能选择的外卖不多。

### 关于充电

充电的地方主要有两个，一个在厕所对面的栏杆上有一个排插，一个在大白工作桌。但是大白那人多的时候最好别去充电，大白他们有几个部门，会吵架
